Problem 60 (Prime pair sets)

Solution detail

Use a standard depth-first backtracking algorithm, where each branch in the search tree represents adding a single prime to the set.

To speed up the search, branches can be pruned early when we know that the resultant sum cannot be smaller than that of some previously-found set of primes.

Finally, we can speed up checking by pre-calculating which pairs of primes concatenate to form another prime, rather than having to check each pair multiple times as we go through the search.
