Problem 53 (Combinatoric selections)

Solution detail

We can use two facts about binomial coefficients to speed up the search:

  • nCr is increasing for 0 ≤ r ≤ n / 2
  • nCn-r = nCr

This means that, for each n, we can iterate up through the values 0 ≤ r ≤ n / 2, and when we find nCr > 1,000,000 for the first time, this means that there are n - 2r + 1 values of r which cause the binomial coefficient to exceed one million.

The values of nCr for increasing r can be calculated using the relation nCr+1 = nCr × (n - r) / (r + 1)
