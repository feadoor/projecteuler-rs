Problem 111 (Primes with runs)

Solution detail

This is just a matter of using a careful implementation. For each digit d, and for each k = 10, 9, 8, ..., 0, generate all numbers containing exactly k occurrences of the digit d. Among all such numbers, find the sum of all the primes - if that sum is non-zero, then add it to a running total and break; otherwise, move onto the next k.
