Problem 36 (Double-base palindromes)

Solution detail

There are fewer base-10 palindromes less than one million than there are binary palindromes. This means that we can simply iterate over the base-10 palindromes, and for each one, check if it is also a binary palindrome.

To iterate over the base-10 palindromes, we can simply iterate over all possible 'first halfs' of numbers, and create palindromes from them by reversing and gluing together.
