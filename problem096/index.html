Problem 96 (Su Doku)

Solution detail

Ignoring the boilerplate of reading in the puzzles from file, the meat of this problem is in quickly finding the solution to a given Sudoku puzzle.

Depth-first search is a prime candidate for this - given a puzzle to solve, fill in the empty cells in order, branching on all digits that are allowed to go into that cell, and backtracking when an impossibility is encountered.

After solving all of the puzzles, adding up to get the solution is a trivial matter.
